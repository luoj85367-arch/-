# 心手相通 可穿戴手语翻译系统
## 项目概述
当前听障群体与健听人沟通存在明显壁垒，传统摄像头手语识别易受光照、遮挡干扰，云端翻译存在网络依赖、高延迟、隐私泄露等问题。本项目基于RDK X5开发柔性可穿戴手语翻译设备，融合柔性传感、IMU姿态采集与边缘AI技术，依靠芯片本地NPU算力完成离线手语识别，实现手语、语音文字双向无障碍交互，无需联网即可低延迟稳定运行，是轻量化助残交互方案。

## 核心功能
1. 搭载12路BNO055九轴IMU采集手部姿态数据，完全规避视觉识别环境干扰问题；
2. 本地离线TCN时序神经网络推理，手语实时转语音文字、语音转文字双向翻译；
3. 拓展手势控制能力，支持手语操控智能家居、无人机，实现多功能复用。

## 核心技术亮点
1. 采用惯性传感替代摄像头，全光照、遮挡场景均可识别，保护用户隐私；
2. RDK X5本地边缘算力，离线运行无网络依赖，大幅降低交互延迟；
3. 柔性轻薄穿戴结构，分体无线传输，佩戴无束缚，适配全天候使用。

## 应用场景
适用于听障人群日常社交、医院政务等公共服务办事、特殊学校手语教学，同时可作为智能家居体感控制、高校嵌入式科创实训载体。

## 系统硬件构成
- 采集终端：ESP32-C3主控 + 双手共12路IMU柔性传感手套，Wi-Fi双通道TCP传输数据
- 计算终端：RDK X5旭日5，本地NPU承载AI模型推理
- 输出模块：语音播报、文字显示外设

## 关键性能参数
- 手语识别准确率94.65%，模型推理帧率40FPS
- 支持35类日常通用手语，无线传输丢包率趋近于0
- 整机离线独立工作，不受网络、明暗环境、手部遮挡限制
