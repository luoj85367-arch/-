# 心手相通 - 手语翻译可穿戴设备
## 项目简介
针对听障人群沟通壁垒，传统视觉手语识别易受遮挡、光照干扰，云端方案延迟高、隐私差。本项目基于RDK X5开发柔性可穿戴手语翻译设备，依托本地NPU离线AI推理，实现手语/语音双向互译，无需联网、抗干扰、低延迟，轻量化助残解决方案。

## 核心功能
1. 12路IMU姿态传感采集手部动作，规避摄像头缺陷
2. 本地离线TCN时序模型推理，手语↔语音文字双向翻译
3. 拓展手势控制智能家居、无人机体感交互

## 核心优势
- 无摄像头：不受光照、衣物遮挡影响，隐私性强
- 边缘离线：依托RDK X5本地算力，脱离网络低延迟运行
- 柔性穿戴：轻薄便携，多场景日常使用
- 多功能复用：兼顾无障碍沟通与人机交互

## 应用场景
听障人士日常社交、医院/政务公共办事、特殊手语教学、居家智能控制、科创实训开发

## 硬件架构
- 采集端：ESP32+C3 + BNO055九轴IMU柔性穿戴手套
- 运算端：RDK X5 本地NPU离线AI推理
- 通信：Wi-Fi双通道TCP数据传输

## 关键指标
- 模型识别准确率94.65%，推理帧率40FPS
- 支持35类常用手语，全环境稳定识别，无线传输无丢包
