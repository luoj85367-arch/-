# 心手相通 —— 可穿戴离线手语采集翻译系统
## 一、项目概述
国内现有约2780万听障人群，在日常社交、就医问诊、政务办事等场景长期存在沟通障碍。目前主流手语识别方案多采用摄像头视觉采集，极易受强光、暗光、衣物遮挡、复杂背景干扰识别效果；云端手语翻译则高度依赖网络，存在交互延迟高、手势画面上传易泄露隐私等短板，市面商用设备普遍价格偏高、便携性不足，难以满足全天候无障碍交流需求。

本项目命名“心手相通”，基于RDK X5旭日5边缘主控芯片，研发柔性可穿戴手语翻译硬件系统。方案摒弃视觉摄像头，采用多通道IMU惯性传感采集手部运动数据，依托芯片内置10TOPS本地NPU算力搭载TCN时序神经网络完成离线AI推理，实现手语与语音文字双向实时互译。设备可完全脱离局域网与公网独立运行，兼顾抗干扰、低延迟、隐私安全、轻量化佩戴等优势，同时拓展体感交互功能，为听障群体提供低成本、高可靠的一站式助残交互解决方案，契合《无障碍环境建设法》无障碍信息化建设相关要求。

## 二、核心功能介绍
1. **无视觉手部姿态采集**
左右手分体式柔性穿戴设备搭载合计12路BNO055九轴IMU传感器，同步采集手部加速度、陀螺仪、姿态四元数等时序运动特征，无需拍摄图像，彻底规避光照、遮挡带来的识别失效问题。
2. **双向离线无障碍翻译**
本地完成全链路AI推理，佩戴者手语动作实时解析转化为文字与语音对外输出；配套收音模块采集健听人群语音并自动转文字展示，搭建听障与健听群体双向对话通道。
3. **多场景体感拓展控制**
突破单一翻译工具定位，内置多协议无线联动逻辑，识别标准化手语语义后可输出控制指令，支持手势操控智能家居设备、无人机飞行，一套硬件同时覆盖助残沟通与科创体感交互。
4. **轻量化无感佩戴设计**
传感模块集成于轻薄柔性基底，无硬质大块结构，不限制手指弯曲、摆动；支持锂电池、Type-C双供电模式，室内外、昼夜各类环境均可长期稳定使用。

## 三、核心技术特点
1. **IMU惯性传感替代传统视觉方案**
依靠惯性运动数据实现手势识别，具备穿透式采集能力，衣物遮挡、极端明暗环境下识别稳定性不受影响，从底层解决视觉识别固有缺陷，同时杜绝图像上传带来的隐私风险。
2. **RDK X5边缘本地离线AI推理**
整套手势采集、数据预处理、模型推理流程全部在本地硬件完成，无需云端转发，大幅降低交互延迟；硬件NPU专门加速TCN时序神经网络，针对连续手语动作时序特征优化识别精度。
3. **双通道无线稳定数据传输**
硬件搭载Wi-Fi双Socket双通道TCP通信，左右手传感数据独立上传并通过时间戳精准同步，自动过滤手部静止无效帧，无线传输丢包率为0，保障手势时序完整不丢失。
4. **完整双向交互闭环逻辑**
串联传感采集、数据降噪、时序模型推理、文字语音输出、外设指令分发全链路，同时支持语音反向转文字，形成手语-文字-语音双向互通的完整交互闭环。

## 四、应用覆盖场景
1. **日常社交场景**：商场、餐厅、街道等公共场合，听障人士可实时完成手语翻译，减少沟通误解与社交隔离；
2. **公共服务场景**：医院问诊、政务大厅、银行业务办理，无需预约专业手语翻译，降低听障群体办事门槛；
3. **特殊教育场景**：辅助特教课堂手语教学，实时判断手势正误，存储动作数据用于课后教学复盘；
4. **居家生活场景**：通过标准化手势操控空调等智能家居，兼顾听障人群与行动不便人群使用；
5. **科创实训场景**：面向嵌入式、机器人爱好者，手语作为无人机无线控制指令，可用于高校教学、科创作品展示。

整体覆盖民生助残、特殊教育、智能家居、科创实训四大领域，受众包含听障群体、特殊教育机构、智能家居使用者、高校嵌入式研发团队。

## 五、系统硬件组成
1. **可穿戴采集终端**：左右两套对称设备，以ESP32-C3为子主控，搭配TCA9548A多路I2C扩展芯片挂载6颗BNO055 IMU，柔性基底集成传感器，负责原始手部运动数据采集与无线上传；内置完整电源管理电路，支持锂电池升压、Type-C自动切换供电。
2. **核心运算单元**：RDK X5旭日5主控芯片，承担数据时序同步、特征归一化预处理、TCN模型NPU离线推理，外接屏幕、麦克风、扬声器、无线拓展模块，统一调度翻译输出与外设控制指令下发。
3. **联动外设模块**：语音输出单元、文字显示屏幕、智能家居通信模组、无人机飞控无线通信模块，接收主控下发的识别结果与控制指令完成多模态输出。
